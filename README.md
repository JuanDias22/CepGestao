# Sistema de Gerenciamento de Endereços

Tecnologias
- ASP.NET MVC
- Entity Framework Core
- SQL Server
- Bootstrap
- JavaScript
- ViaCEP API

Funcionalidades
- Login de usuários
- CRUD de endereços
- Integração ViaCEP
- Exportação CSV
- Sessão de autenticação

Como executar
1. Clone o repositório
2. Execute Update-Database
3. Rode o projeto
